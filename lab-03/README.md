# Лабораторна робота №3. Візуалізація багатовимірних даних

## Мета роботи
Ознайомитись з бібліотеками для візуалізації даних у Python (Matplotlib, Seaborn). Навчитися очищувати дані від пропусків та будувати складні багатовимірні графіки для аналізу залежностей, розподілів і кореляцій у датасеті.

## Обраний датасет
Для виконання роботи використано **Automobile Data Set** (з репозиторію машинного навчання UCI).
- **Тип даних:** Багатовимірні (Multivariate).
- **Атрибути:** Категоріальні, цілі та дійсні числа.
- **Очищення (Data Cleaning):** Виявлено та видалено рядки з відсутньою цільовою змінною (`price`), інші числові пропуски заповнено медіаною, а категоріальні — модою.

## Побудовані візуалізації (8 графіків)
1. **Linear Regression Scatter Plot:** Лінійна залежність ціни автомобіля від об'єму двигуна з відображенням довірчого інтервалу.
2. **Polynomial Regression Plot:** Поліноміальна залежність витрати палива в місті від потужності (кінських сил).
3. **Histogram (5 Bins):** Розподіл автомобілів за 5 суворими ціновими діапазонами з накладанням ліній середнього значення та медіани.
4. **Correlation Heatmap:** Теплова матриця кореляції числових атрибутів (відображено лише нижній трикутник для чистоти сприйняття).
5. **Boxplot + Stripplot:** Аналіз розподілу цін та пошук викидів за типом кузова з накладанням точок кожного окремого автомобіля.
6. **Multidimensional Pairplot:** Попарна матриця розсіювання ключових характеристик з урахуванням типу палива (засновано на прикладах зі статей Medium).
7. **Barplot with Annotations:** Стовпчаста діаграма середньої вартості автомобілів у розрізі брендів із точними текстовими підписами сум.
8. **Violinplot + Stripplot:** Ядерна оцінка густини розподілу цін залежно від типу приводу автомобіля.
