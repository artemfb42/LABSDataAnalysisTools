\# Лабораторна робота №2: Підготовчий етап



\##  Опис завдання

Ця лабораторна робота присвячена основам збору, очищення та аналізу даних.

\- \*\*Частина 1:\*\* Завантаження файлів VHI-індексу для адміністративних одиниць України через модуль `urllib`, очищення даних та виконання вибірок за допомогою `pandas`.

\- \*\*Частина 2:\*\* Аналіз датасету споживання електроенергії (Individual Household Electric Power Consumption), складна фільтрація, стандартизація/нормалізація, One Hot Encoding та профілювання часу виконання процедур за допомогою модуля `timeit`.



\## Вимоги до системи

\* \*\*Операційна система:\*\* Windows / macOS / Linux

\* \*\*Версія інтерпретатора:\*\* Python 3.9 або новіше

\* \*\*Менеджер пакетів:\*\* `pip`

\* \*\*Залежності:\*\* Всі необхідні бібліотеки (`pandas`, `scikit-learn`, `jupyter` тощо) вказані у файлі `requirements.txt`.
